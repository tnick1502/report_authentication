Данный проект разработан командой компании "МОСТДОРГЕОТРЕСТ" с целью обеспечения гарантий подлинности данных.


GEOREPORT позволяет аутентифицировать протоколы лабораторных испытаний, тем самым создавая дополнительную степень защиты.

Описание проекта

Данный проект позволяет дублировать данные протоколов испытаний на специальном серевере исключая ручные изменений в протоколе вне лаборатории.


Суть работы:

  • При выдаче протокола испытания результаты заносятся на сервер приложения, также сервер выдает специальные QR-код с доступом к этим данным.

  • QR-код помещается обратно на протокол испытания, тем самым связывая данные на сервере с протоколом. В случае иеменения протокола редоктором, данные на сервере останутся без изменений.

  • Для проверки подлинности данных наведите камеру телефона на QR-код, перейдите по зашифрованной в нем ссылке и сверьте полученные данные с данными в протоколе испытания.
